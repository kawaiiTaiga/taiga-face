# taiga

Monorepo for TAIGA experiments.

## Projects

- `taiga-face/` - browser-based face renderer and formula DSL runtime
- `signage/` - browser-based reactive text signage simulator

## Quick Start

### taiga-face

```bash
cd taiga-face
npm install
npm start
```

Runs the face runtime on the local development server.

### signage

Open `signage/index.html` directly in a browser, or serve the repository root with any static file server and open `signage/index.html`.
